# perception 模块

## 输入
- 页面处理对象（提供 HTML、URL、截图能力）。
- 用户目标字符串（可选）。

## 输出
- 页面结构化结果：
  - `page_schema`：`PageSchema` 字典结构。
  - `vlm_hints`：可选视觉补充字段。

## 职责
- 解析 DOM 并提取可交互元素。
- 生成稳定的页面结构表示。
- 注入 JavaScript 进行视口过滤、交互语义剪枝和绝对坐标提取。
- 在页面顶层动态画框标号（SoM），截取带标号截图，并在截图后立即清理现场恢复页面原貌。
- 调用执行层映射注册并将 `PageSchema.element_id` 同步为持久化 ID。
- 过滤无法注册选择器的元素，保证输出元素均可执行。

## 禁止做的事情
- 禁止生成动作决策。
- 禁止直接调用执行动作。
- 禁止改写 workflow/llm 层输入输出协议。
- 禁止输出未注册的 `element_id`。
